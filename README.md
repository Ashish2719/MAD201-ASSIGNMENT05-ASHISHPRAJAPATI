## Description
This is a Task Manager mobile application built with React Native and Expo. It allows users to create, read, update, and delete (CRUD) daily tasks. The app uses the **Context API** for global state management and combines **Stack and Tab Navigation** for a complete user experience.

## Features
* **Add Tasks:** Input title and description to create new tasks.
* **View Tasks:** See all tasks or filter to see only "Completed" tasks via Tabs.
* **Task Details:** Click a task to view details, mark as complete, or delete.
* **Navigation:** Seamless movement between screens using Expo Router.

## How to Run
1.  Open the terminal in the project folder.
2.  Run `npm install` to install dependencies.
3.  Run `npm run web` to launch the app.
